Backbone Extraction¶

Extracts the Backbone edges from the original graph and the distance closure computation.

backbone.backbone(A, C)¶

Return backbone edges based on the the original graph and its transitive closure. By definition, the backbone are the edges that did not change value in the transitive closure computation.

Parameters:
  • A (matrix) – Adjacency matrix from original graph.
  • C (matrix) – Adjacency matrix from transitive closure graph
Returns:

B – Adjacency matrix where backbone edges are 1, semi-metric edges are 2, diagonal is -1, 0 otherwise.
Return type:

array

Note

Inputs accepts both dense and sparse matrices.

Examples

>>> C = transitive_closure_numpy(A, kind='metric')
>>> B = backbone_numpy(A, C)

You can then access the backbone edges by using:

>>> import numpy as np
>>> rows, cols = np.where(B==1)
