Backbone extraction¶

Extracts the Backbone edges from the original graph and the distance closure computation.

distanceclosure.backbone.backbone(A, C)¶

Return backbone edges based on the the original graph and its transitive closure. By definition, the backbone are the edges that did not change value in the transitive closure computation.

Parameters:
  • A (dense or sparse matrix) – Adjacency matrix from original graph.
  • C (dense or sparse matrix) – Adjacency matrix from transitive closure graph
Returns:

Adjacency matrix where backbone metric edges are 1 and semi-metric edges are 2. On the dense matrix, the diagonal is -1 and 0 otherwise.
Return type:

B (dense or sparse matrix)

Examples

>>> # Dense Matrix
>>> C = transitive_closure(A, kind='metric', algorithm='dense')
>>> B = backbone(A, C)

You can then access the backbone metric edges by using:

>>> import numpy as np
>>> rows, cols = np.where(B==1)

>>> # Sparse Matrix
>>> C = transitive_closure(A, kind='metric', algorithm='dijkstra')
>>> B = backbone(A, C)
